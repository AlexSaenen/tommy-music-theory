Scales

A scale is a set of notes, usually spanning one octave. Scales are also typically the same set of notes ascending and descending, but they could be written differently depending on which direction you are going.

Many melodies and chord progressions are built off of a single scale.

Depending on the number of notes in the scale, the scale can be described differently. For instance, an eight note scale would be called an octatonic scale, a seven note scale would be called a heptatonic scale, and a five note scale would be called a pentatonic scale.

You can build scales using a combination of whole steps, half steps, and intervals.

One example of a scale is called the chromatic scale, and it is the only dodecatonic (12-note) scale. The chromatic scale is built entirely of half steps. If you play half steps from C to the C an octave higher you will get a C chromatic scale. The set of notes in a C chromatic scale ascending is: [C, C#, D, D#, E, F, F#, G, G#, A, A#, B]. Descending the scale would be [C, B, Bb, A, Ab, G, Gb, F, E, Eb, D, Db], which is the same set of notes written differently. Another example is the whole tone scale, which belongs to the hexatonic (six-note) scale family. A whole-tone scale is built entirely of whole steps. If you play only whole steps from C to C an octave higher, you will get the C whole tone scale. Ascending the C whole tone scale would be: [C, D, E, F#, G#, A#]. Descending the scale would be: [C, Bb, Ab, Gb, E, D], which is once again the same set of notes written differently.
